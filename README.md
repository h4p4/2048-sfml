# 2048-sfml
If you can build it you're a god.

## Description

Simple recreation of [2048 game](https://play2048.co/)

### Dependencies

* SFML dlls
* [GCC 7.3.0 MinGW (SEH) - 64-bit](https://www.sfml-dev.org/download/sfml/2.5.1/)
* Windows 10

## Getting Started

* Put SFML dlls in the same place as executable.

### Executing program

* Good luck
```
g++ something
```
